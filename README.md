# 💰 Personal Finance Tracker

A console-based finance tracker built in Python that allows users to log expenses, categorize transactions, and view spending summaries.

## 🚀 Features

- Add income and expenses
- Categorize transactions
- View monthly summaries
- Track total balance
- Store data using JSON files

## 🛠️ Technologies Used

- Python 3
- JSON
- Object-Oriented Programming (OOP)

## 📂 Project Structure

```text
finance-tracker/
│
├── main.py
├── transactions.json
└── README.md
```

## ▶️ How to Run

```bash
python main.py
```
